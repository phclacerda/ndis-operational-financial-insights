# NDIS Operational & Financial Insights Dashboard (Excel)

This project simulates an operational and financial reporting scenario in the NDIS context using Microsoft Excel.

## Objective
Provide clear, actionable insights on service delivery, utilisation, financial performance and operational risk to support decision-making.

## Tools & Techniques
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- XLOOKUP, FILTER, IF formulas
- Basic data modelling

## Key Insights
- Delivered funding and utilisation by region
- Budget status (on-track vs overspent participants)
- Revenue, cost and margin by support category
- SLA expiry risk tracking for proactive follow-up

## Output
- Interactive Excel dashboard
- Stakeholder-ready PDF summary

## Notes
All data used in this project is fictitious and created for demonstration purposes only.
